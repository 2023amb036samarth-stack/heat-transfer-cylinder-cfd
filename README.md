# heat-transfer-cylinder-cfd

# CFD Simulation of Heat Transfer Over Cylinder Using OpenFOAM

This project simulates external flow over a heated cylinder using OpenFOAM and post-processing in ParaView.

The study focuses on velocity distribution, temperature field evolution, drag estimation and flow separation behavior.

---

## Objective

To analyze fluid flow and convective heat transfer characteristics over a circular cylinder.

---

## Software Used

OpenFOAM  
ParaView  

---

## Parameters Investigated

Velocity field (Ux component)

Temperature distribution

Flow separation location

Drag coefficient estimation

Wake formation behind cylinder

---

## Post-Processing

Performed using ParaView:

• velocity contours  
• temperature contours  
• line probe velocity extraction  
• separation region visualization  

---

## Applications

External aerodynamics

Heat transfer analysis

Wake flow behaviour

CFD learning workflow development
